# API Projects

This repository contains four independent projects that consume external APIs.

## Projects

### 1. API_Mapas
Geolocation project that displays locations on a map.

### 2. API_Redes sociales
Social media API project that displays the profile of a GitHub account.

### 3. API_Streming
Streaming API project that obtains information about a movie from a movie application.

### 4. API_base de datos
Database/Open Data API project that retrieves and displays external data.

## Installation

1. Clone the repository.
2. Open the project folder you want to run.
3. Install dependencies:

npm install

4. Run the project:

npm start
